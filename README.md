# Luxury Watch Store Website

A multi-page luxury watch showcase website built using **HTML, CSS, and JavaScript**.  

The website demonstrates fundamental front-end development skills including webpage structure, styling, interactivity, and multi-page navigation.

---

## Project Overview

This project is a static website designed for a watch store that sells various types of watches. The website provides users with information about products, blog articles about watches, research references, and details about the development team.

The project focuses on applying core web development technologies to create a **user-friendly and visually appealing interface**.

---

## Features

- Multi-page website structure
- Responsive layout using CSS
- Interactive components using JavaScript
- Product showcase with watch images
- Informational blog page
- Research comparison page
- About Us page with team details
- Navigation bar with hover effects
- Form validation using JavaScript
- Image slider functionality

---

## Website Pages

### Home Page
- Website introduction
- Featured watch products
- Promotional sections
- Navigation links to other pages

### Product Page
- Displays different watch brands
- Product images and descriptions
- Interactive product elements

### Blog Page
- Articles about watch brands and styles
- Informational content for users

### Research Page
- Comparison of reference websites
- Analysis of design inspiration and technologies

### About Us Page
- Information about the development team
- Contact and feedback section

---

## Technologies Used

- **HTML5** – Structure of the webpages
- **CSS3** – Styling and layout design
- **JavaScript** – Interactive features and validation
- **Media Assets** – Images and video content

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/dawagrg/BrosWatch.git
```

2. Open the project folder

3. Run the website by opening:

```
home.html
```

in any modern web browser.
